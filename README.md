# Emma & Rob — Where Next?

Interactive map of 9 cities we're considering for our next chapter.

## How to view

Open `index.html` in any browser. No build step needed.

## Features

- Dark CartoDB tile layer
- Markers sized and colored by sunny days (warmer/larger = more sun)
- Clickable sidebar that flies to each city
- Styled dark popups with all city data
- Compass rose
- Mobile responsive
- Leaflet.js, no dependencies beyond CDN
